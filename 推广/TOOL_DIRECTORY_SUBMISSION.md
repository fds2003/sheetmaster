## 二、平台针对性提交战术

### 1. AlternativeTo (最高优先级：精准截流)
*   **战术**：碰瓷营销
*   **步骤**：
    1. 注册账号并提交 SheetMaster 基础信息。
    2. 搜索付费/AI 竞品："GPTExcel", "FormulaBot", 或 "Ajelix"。
    3. 点击竞品页面的 **"Suggest Alternative"**，添加 SheetMaster。
    4. **推荐理由 (Pros)**：勾选/填写 **"Free"**, **"No Signup Required"**，并强调 **"More precise than AI"**。

### 2. Capterra & G2 (B2B 权威外链)
*   **战术**：获取 Dofollow 权威链接
*   **步骤**：
    1. 申请入驻（建议使用 `@getsheetmaster.com` 企业邮箱）。
    2. 分类选择：**"Data Analysis Tools"** 或 **"Spreadsheet Software"**。
    3. 审核通过后，邀请 2-3 个真实用户留 5 星好评（评价提及 "Saved me hours on VLOOKUP" 等字眼）。

### 3. Slant.co (极客与对比社区)
*   **战术**：问答推荐
*   **步骤**：
    1. 搜索相关问题（如 "Best Excel Add-ins" 或 "Best tools for data cleaning"）。
    2. 若有问题，将 SheetMaster 作为 **Option** 添加。
    3. 若无问题，自问自答："What is the best free alternative to AI formula generators?"。
    4. 列出 **Pros**："Learn as you generate"。

### 4. 普通工具导航站 (Tools.buzz, ToolFinder 等)
*   **战术**：批量铺量
*   **步骤**：使用上方的"标准提交资料包"批量填表，等待 1-2 周上线。
